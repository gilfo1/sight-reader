# User Settings Guide

All settings are located in the "Settings" accordion at the top of the page. Here is a breakdown of what each one does.

## Layout Settings

- **Measures per Line**: Choose how many measures to display on a single line of music (1 to 8).
- **Lines**: Set the total number of lines to display at once (1 to 10).
- **Staff Type**:
    - **Grand Staff**: Standard for piano (Treble and Bass clefs).
    - **Treble Clef**: Only display the treble clef.
    - **Bass Clef**: Only display the bass clef.

## Generation Controls

- **Notes per Beat**: Controls the number of notes stacked in each beat (1 to 10).
    - Setting this to 1 generates single melodies.
    - Setting it to 2 or more generates intervals and chords.
    - In Grand Staff mode, notes are intelligently distributed between both hands.
- **Min Note / Max Note**: Defines the absolute pitch range for generated notes.
    - The available options adjust dynamically based on the selected Staff Type.
- **Note Values**: Check the durations you want to practice. You can mix and match Whole, Half, Quarter, Eighth, and Sixteenth notes.
- **Key Signatures**: Select one or more major keys. The app will randomly pick from your selections for each line.
    - **Chromatic**: Enable this to add random accidentals outside of the selected key signature for a more difficult reading experience.

## MIDI & Feedback

- **MIDI Status**: Shows the name of the connected MIDI device and its connection state (Green = Connected, Red = Disconnected).
- **Show MIDI Notes**: Open this section to see a text display of the notes you are currently pressing on your keyboard.
